Leave Request Management System (LRMS)

Leave Request Management System, a terminal-based application developed in C. This project was a fantastic opportunity to dive deep into system design, file handling, and user management while creating a practical tool for organizations to manage employee leave requests efficiently.

‚ú® What does it do?

The system allows Employees to request leave, track their leave history, and update their profiles, while Managers can approve/reject requests, review pending sign-ups, and generate detailed leave statistics. It also includes features like secure password handling, password recovery, and cross-platform compatibility.

I wanted to challenge myself to build a functional, real-world application using core programming concepts. It was a great way to strengthen my skills in C programming, problem-solving, and system design while creating something that could genuinely be useful in a professional setting.
